# TeXposit

If you are looking for the TeXposit editor, please go to [app.texposit.com](https://app.texposit.com). If you want to learn about TeXposit, our website is [texposit.com](https://texposit.com).  

Feel free to use the Issues page to report bugs or submit requests about TeXposit, but you can also reach us directly at contact@texposit.com.

In the future, we will use this repository to host the TeXposit source code.
